# learningGit
It's a small repository to learn me how to use GitHub and Git.

## How to execute the java code?

*Prerequisites :* 
- JDK installed

1. Download the LearningGit.java file
2. Open a command prompt window and go to the directory where you saved the java program (LearningGit.java).
3. Type 'javac LearningGit.java' and press enter to compile the code.
4. Now, type 'java LearningGit' to run the program.
5. You will be able to see the result printed on the window.
